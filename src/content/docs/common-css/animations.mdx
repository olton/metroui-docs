The Animation Utilities in Metro UI provide a comprehensive set of CSS classes for adding predefined animations to elements, with support for hover effects and customizable timing.

## Animation Classes

| Class | Hover Class | Description |
| --- | --- | --- |
| `.ani-spin` | `.ani-hover-spin` | Rotates element 360 degrees clockwise |
| `.ani-spin-reverse` | `.ani-hover-spin-reverse` | Rotates element 360 degrees counter-clockwise |
| `.ani-pulse` | `.ani-hover-pulse` | Pulsing rotation effect |
| `.ani-spanner` | `.ani-hover-spanner` | Wrench/spanner-like swinging motion |
| `.ani-ring` | `.ani-hover-ring` | Bell ringing animation |
| `.ani-vertical` | `.ani-hover-vertical` | Vertical bouncing motion |
| `.ani-horizontal` | `.ani-hover-horizontal` | Horizontal shaking motion |
| `.ani-flash` | `.ani-hover-flash` | Flashing visibility |
| `.ani-bounce` | `.ani-hover-bounce` | Bouncing up and down |
| `.ani-float` | `.ani-hover-float` | Gentle floating motion |
| `.ani-heartbeat` | `.ani-hover-heartbeat` | Pulsating like a heartbeat |
| `.ani-shake` | `.ani-hover-shake` | Shaking motion (alias for wrench) |
| `.ani-shuttle` | `.ani-hover-shuttle` | Scaling and rotating motion |
| `.ani-pass` | `.ani-hover-pass` | Element passes through from left to right |
| `.ani-ripple` | `.ani-hover-ripple` | Expanding ripple effect |

## CSS Variables for Animation Timing

The animation system provides CSS variables for consistent timing across your application:

### Duration Variables

| Variable | Value |
| --- | --- |
| `--base-duration-0` | 0ms |
| `--base-duration-100` | 100ms |
| `--base-duration-200` | 200ms |
| `--base-duration-300` | 300ms |
| `--base-duration-400` | 400ms |
| `--base-duration-500` | 500ms |
| `--base-duration-600` | 600ms |
| `--base-duration-700` | 700ms |
| `--base-duration-800` | 800ms |
| `--base-duration-900` | 900ms |
| `--base-duration-1000` | 1s |

### Easing Variables

| Variable | Value |
| --- | --- |
| `--base-easing-linear` | cubic-bezier(0, 0, 1, 1) |
| `--base-easing-easeIn` | cubic-bezier(0.7, 0.1, 0.75, 0.9) |
| `--base-easing-easeOut` | cubic-bezier(0.3, 0.8, 0.6, 1) |
| `--base-easing-easeInOut` | cubic-bezier(0.6, 0, 0.2, 1) |

## Usage Examples

### Basic Animation

Apply animations directly to elements:

```html
<div class="ani-spin">
    <span class="mif-spinner"></span>
</div>

<div class="ani-bounce">
    <span class="mif-bell"></span>
</div>
```

### Hover Animations

Animation starts only when user hovers over the element:

```html
<div class="ani-hover-ring">
    <span class="mif-bell"></span>
</div>

<button class="button ani-hover-shuttle">
    Hover me
</button>
```

### Customizing Animation with CSS

You can customize the animation properties using inline styles or CSS:

```html
<div class="ani-spin" style="animation-duration: var(--base-duration-500); animation-timing-function: var(--base-easing-easeInOut);">
    Custom speed spinner
</div>
```

```css
.custom-animation {
    animation-duration: 2.5s;
    animation-timing-function: var(--base-easing-easeOut);
    animation-delay: 0.5s;
}
```

## Additional Animation Keyframes

The animation system also includes the following keyframe animations that you can use with custom CSS:

| Keyframe Name | Description |
| --- | --- |
| `swinging` | Pendulum-like swinging animation |
| `scaleout` | Scale from 0 to 1 with fade out |
| `cubemove` | Cube movement with rotation |
| `orbit` | Orbital rotation with opacity changes |
| `metro-slide` | Slide from left to right |
| `metro-opacity` | Fade in animation |
| `ani-shrink` | Shrink element to 50% of size |
| `ani-pre-spin` | Simple 360-degree spin |
| `ani-bg-stripes` | Moving background stripes |

Example of using a custom keyframe animation:

```css
.custom-element {
    animation: orbit 2s linear infinite;
}
```

## Combining with Other Components

These animations work well with icon fonts, buttons, alerts, and other UI components:

```html
<!-- Animated loading indicator -->
<div class="loading-indicator">
    <span class="mif-spinner ani-spin"></span> Loading...
</div>

<!-- Attention grabbing button -->
<button class="button primary ani-hover-pulse">
    Important Action
</button>

<!-- Alert with animation -->
<div class="alert warning ani-horizontal">
    <span class="mif-warning"></span> Warning message
</div>
```
