The Flex Utilities in Metro UI provide a comprehensive set of classes for working with Flexbox layouts. These utilities allow you to easily create flexible, responsive layouts without writing custom CSS.

## Display Flex

Basic classes for enabling flex container behavior:

| Class | Description |
| --- | --- |
| `.d-flex` | Creates a flex container (display: flex) |
| `.d-inline-flex` | Creates an inline flex container (display: inline-flex) |

## Flex Direction

Control the direction of flex items:

| Class | Description |
| --- | --- |
| `.flex-row` | Items are placed in a row (default) |
| `.flex-row-reverse` | Items are placed in a row in reverse order |
| `.flex-column` | Items are stacked vertically |
| `.flex-column-reverse` | Items are stacked vertically in reverse order |

## Flex Wrap

Control how flex items wrap:

| Class | Description |
| --- | --- |
| `.flex-nowrap` | Items will not wrap (default) |
| `.flex-wrap` | Items will wrap if needed |
| `.flex-wrap-reverse` | Items will wrap in reverse if needed |

## Alignment: Align Items

Align flex items along the cross axis:

| Class | Description |
| --- | --- |
| `.flex-align-items-start` | Items are aligned to the start of the cross axis |
| `.flex-align-items-end` | Items are aligned to the end of the cross axis |
| `.flex-align-items-center` | Items are centered on the cross axis |
| `.flex-align-items-baseline` | Items are aligned by their baselines |
| `.flex-align-items-stretch` | Items are stretched to fill the container (default) |

## Alignment: Justify Content

Align flex items along the main axis:

| Class | Description |
| --- | --- |
| `.flex-justify-content-start` | Items are packed toward the start |
| `.flex-justify-content-end` | Items are packed toward the end |
| `.flex-justify-content-center` | Items are centered along the line |
| `.flex-justify-content-between` | Items are evenly distributed with first item at the start and last item at the end |
| `.flex-justify-content-around` | Items are evenly distributed with equal space around them |
| `.flex-justify-content-evenly` | Items are distributed so that spacing between any two items is equal |

## Individual Item Alignment

Control alignment for individual flex items:

| Class | Description |
| --- | --- |
| `.flex-align-self-auto` | Uses parent's align-items value (default) |
| `.flex-align-self-start` | Item is aligned to the start |
| `.flex-align-self-end` | Item is aligned to the end |
| `.flex-align-self-center` | Item is centered |
| `.flex-align-self-baseline` | Item is aligned by its baseline |
| `.flex-align-self-stretch` | Item is stretched to fill the container |

## Growth and Shrinking

Control how flex items grow and shrink:

| Class | Description |
| --- | --- |
| `.flex-grow` | Makes all child items grow to fill available space |
| `.flex-no-grow` | Prevents all child items from growing |
| `.flex-shrink` | Allows all child items to shrink if necessary |
| `.flex-no-shrink` | Prevents all child items from shrinking |
| `.flex-grow-self` | Makes the item grow to fill available space |
| `.flex-no-grow-self` | Prevents the item from growing |
| `.flex-shrink-self` | Allows the item to shrink if necessary |
| `.flex-no-shrink-self` | Prevents the item from shrinking |

## Auto Margins for Alignment

Use auto margins for easy alignment:

| Class | Description |
| --- | --- |
| `.flex-left` | Pushes item(s) to the left |
| `.flex-right` | Pushes item(s) to the right |
| `.flex-top` | Pushes item(s) to the top |
| `.flex-bottom` | Pushes item(s) to the bottom |

## Gap Between Items

Control spacing between flex items:

| Class | Description |
| --- | --- |
| `.gap-0` | No gap between items |
| `.gap-1` | 4px gap between items |
| `.gap-2` | 8px gap between items |
| `.gap-3` | 12px gap between items |
| ... | ... |
| `.gap-12` | 48px gap between items |

## Order Control

Change the visual order of specific flex items:

| Class | Description |
| --- | --- |
| `.order-0` | Sets order: 0 |
| `.order-1` | Sets order: 1 |
| `.order-2` | Sets order: 2 |
| ... | ... |
| `.order-24` | Sets order: 24 |

## Equal-Width Items

Make all items take equal width:

| Class | Description |
| --- | --- |
| `.flex-equal-items` | Makes all direct children take equal width |

## Centering Helper

Easily center content both horizontally and vertically:

| Class | Description |
| --- | --- |
| `.flex-center` | Centers items both horizontally and vertically |

## Responsive Variants

All flex utilities have responsive variants that apply at specific breakpoints and above:

| Breakpoint Suffix | Screen Width |
| --- | --- |
| `-xs` | ≥ 0px (Extra small screens) |
| `-sm` | ≥ 576px (Small screens) |
| `-md` | ≥ 768px (Medium screens) |
| `-lg` | ≥ 992px (Large screens) |
| `-xl` | ≥ 1200px (Extra large screens) |
| `-xxl` | ≥ 1452px (Extra extra large screens) |

Example usage:
- `.flex-column-md` - Stack items vertically on medium screens and above
- `.flex-justify-content-between-lg` - Apply space-between on large screens and above
- `.flex-align-items-center-sm` - Center items on small screens and above

## Usage Examples

### Basic Flex Container

```html
<div class="d-flex gap-2">
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
</div>
```

### Responsive Layout Changes

```html
<div class="d-flex flex-column flex-row-md gap-2">
    <div>Stacked on mobile, side by side on tablet and above</div>
    <div>Second item</div>
</div>
```

### Alignment and Spacing

```html
<div class="d-flex flex-justify-content-between flex-align-items-center">
    <div>Left aligned</div>
    <div>Center aligned vertically, right aligned horizontally</div>
</div>
```

### Equal Width Columns with Responsive Behavior

```html
<div class="d-flex flex-column flex-row-md flex-equal-items gap-2">
    <div>Equal width on tablet and above</div>
    <div>Equal width on tablet and above</div>
    <div>Equal width on tablet and above</div>
</div>
```

### Item Ordering

```html
<div class="d-flex">
    <div class="order-3">Appears third</div>
    <div class="order-1">Appears first</div>
    <div class="order-2">Appears second</div>
</div>
```

### Pushing Items to Edges

```html
<div class="d-flex">
    <div>Left item</div>
    <div class="flex-right">Right item</div>
</div>
```

### Perfect Centering

```html
<div class="flex-center" style="height: 200px;">
    <div>Perfectly centered both horizontally and vertically</div>
</div>
```
