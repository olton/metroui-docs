The Table Component is a powerful and feature-rich data table for displaying, sorting, filtering, and manipulating tabular data. 
It supports pagination, searching, row selection, and many other features.

## Usage

### Basic Table

You can convert any HTML table into a Metro Table by adding the `data-role="table"` attribute.

```html
<table data-role="table">
    ...
</table>
```

or load data from a JSON file using the `data-source` attribute.

```html
<div class="example" lang="ua">
        <table class="table striped row-hover border responsive-md fixed-header"
               data-role="table"
               data-caption="Table caption"
               data-rows="50"
               data-rows-steps="5, 10, 50, 100"
               data-show-activity="false"
               data-source="data/table.json"
               data-show-skip="true"
               data-pagination-wrapper="#pagination"
               data-pagination-island-size="1"
               data-pagination-short-track="5"
               data-skip-wrapper="#skip"
               data-check="true"
               data-check-style="2"
               data-use-current-slice="true"
               data-show-inspector-button="true"
               data-horizontal-scroll="true"
               data-horizontal-scroll-stop="xxl"
               data-cell-wrapper="true"
        >
            <thead id="head1"></thead>
            <tbody id="body1"></tbody>
        </table>
        <div class="row">
            <div class="cell-md-8">
                <div id="pagination"></div>
            </div>
            <div class="cell-md-4">
                <div id="skip"></div>
            </div>
        </div>
    </div>
```

## API Methods

The Table Component provides the following methods:

### reload()

Reloads the table data from the source.

```javascript
const table = Metro.getPlugin('#table', 'table');
table.reload();
```

### setData(data)

Sets new data for the table.

```javascript
const table = Metro.getPlugin('#table', 'table');
table.setData([
  { id: 1, name: "John", age: 30 },
  { id: 2, name: "Jane", age: 25 }
]);
```

### setHeadData(data)

Sets new header data for the table.

```javascript
const table = Metro.getPlugin('#table', 'table');
table.setHeadData([
  { name: "id", title: "ID" },
  { name: "name", title: "Name" },
  { name: "age", title: "Age" }
]);
```

### setSource(source)

Sets a new data source URL.

```javascript
const table = Metro.getPlugin('#table', 'table');
table.setSource('data/new-table-data.json');
```

### search(val)

Performs a search on the table data.

```javascript
const table = Metro.getPlugin('#table', 'table');
table.search("John");
```

### getSelectedItems()

Returns the selected items.

```javascript
const table = Metro.getPlugin('#table', 'table');
const selected = table.getSelectedItems();
```

### getFilteredItems()

Returns the filtered items.

```javascript
const table = Metro.getPlugin('#table', 'table');
const filtered = table.getFilteredItems();
```

### getAllItems()

Returns all items.

```javascript
const table = Metro.getPlugin('#table', 'table');
const allItems = table.getAllItems();
```

### getItem(index)

Returns the item at the specified index.

```javascript
const table = Metro.getPlugin('#table', 'table');
const item = table.getItem(0);
```

### addItem(item)

Adds a new item to the table.

```javascript
const table = Metro.getPlugin('#table', 'table');
table.addItem({ id: 3, name: "Bob", age: 40 });
```

### updateItem(index, item)

Updates an item at the specified index.

```javascript
const table = Metro.getPlugin('#table', 'table');
table.updateItem(0, { id: 1, name: "John Updated", age: 31 });
```

### deleteItem(index)

Deletes an item at the specified index.

```javascript
const table = Metro.getPlugin('#table', 'table');
table.deleteItem(0);
```

## Example Usage

### HTML

```html
<table id="table" data-role="table"
       data-source="data/table-data.json"
       data-rows="10"
       data-check="true"
       data-rownum="true">
</table>
```

### JavaScript

```javascript
// Initialize with JavaScript
Metro.makePlugin('#table', 'table', {
    source: 'data/table-data.json',
    rows: 10,
    check: true,
    rownum: true,
    onRowClick: function(row) {
        console.log('Row clicked:', row);
    }
});
```

## Data Format

The Table Component expects data in the following format:

```javascript
{
  "head": [
    {
      "name": "id",
      "title": "ID",
      "sortable": true
    },
    {
      "name": "name",
      "title": "Name",
      "sortable": true
    },
    {
      "name": "age",
      "title": "Age",
      "sortable": true
    }
  ],
  "body": [
    {
      "id": 1,
      "name": "John",
      "age": 30
    },
    {
      "id": 2,
      "name": "Jane",
      "age": 25
    }
  ]
}
```

## Column Options

Each column in the table can be configured with the following options:

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| title | String | undefined | Column title |
| format | String | undefined | Format for cell values |
| formatMask | String | undefined | Format mask for cell values |
| name | String | undefined | Column name |
| colspan | Number | null | Column span |
| size | String | undefined | Column size |
| sortable | Boolean | false | Enable sorting for this column |
| sortDir | String | undefined | Sort direction (asc/desc) |
| clsColumn | String | undefined | CSS class for the column |
| cls | String | undefined | CSS class for cells in this column |
| show | Boolean | true | Show/hide the column |
| required | Boolean | true | Column is required |
| field | String | undefined | Field name for form inputs |
| fieldType | String | undefined | Field type for form inputs |
| validator | Function | undefined | Validator function for form inputs |
| template | String | undefined | Template for cell rendering |
| type | String | "data" | Column type |
