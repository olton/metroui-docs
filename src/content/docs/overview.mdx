<Leader content="Welcome to Metro UI - popularity framework for creating websites!"/>

Metro UI is a free, open source, html-first toolkit for developing reactive websites with HTML, CSS, and JS. 
With Metro UI, you can easily and quickly make a site from prototype to production.

Metro UI includes general styles and JavaScript routines, responsive grid, layouts, typography, 150+ ready to use components, and 800+ built-in icons.
Also Metro UI includes a set of libraries for working with dates and times, colors, strings, html, and others, including a model to create reactive pages and router for SPA.

## Features:

+ **Declarative**: Metro UI is declarative, meaning you can create complex interfaces with simple HTML attributes. No need to write JavaScript for basic interactions.
+ **Responsive**: Metro UI is fully responsive, adapting to different screen sizes and orientations. It includes a responsive grid system and media queries.
+ **Customizable**: Metro UI is highly customizable, allowing you to change colors, sizes, and styles using CSS variables. You can also create your own themes.
+ **Libraries**: Metro UI includes nine JavaScript libraries to work with HTML elements, strings, color, datetime, routing, and data, including hooks and reactivity.
+ **Common Styles**: Metro UI provides a set of common styles for typography, flexbox, spacing, sizing, and other elements, ensuring a consistent look and feel across your application.
+ **Common JavaScript Routines**: Metro UI includes common JavaScript routines for making your work easy.
+ **Responsive Grid**: Metro UI includes a responsive grid system that allows you to create flexible layouts with ease.
+ **150+ Components**: Metro UI includes 150+ components for any purposes.
+ **Dark & Light**: Metro UI supports dark-light switching out of the box, including the relevant components.
+ **Metro Icon Font**: includes 800+ icons carefully selected for any purpose.
+ **Dependencies Free**: Metro UI has no dependencies, so you can use it with any framework or library.
